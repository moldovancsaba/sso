# Kiro Agent Skill

Connect to Kiro AI assistant for advanced coding, analysis, and task execution.

## Description

This skill allows Clawdbot to communicate with Kiro, providing access to:
- Advanced code analysis and generation
- File system operations
- Complex problem solving
- Multi-step task execution
- Repository exploration and management

## Usage

Use this skill when you need:
- Complex coding tasks
- File analysis or modification
- Advanced problem solving
- Multi-step workflows
- Repository management

## Commands

- `kiro ask <question>` - Ask Kiro a question
- `kiro code <task>` - Request code generation or analysis
- `kiro fix <problem>` - Ask Kiro to fix an issue
- `kiro analyze <file/topic>` - Deep analysis of files or topics

## Requirements

- Kiro IDE running and accessible
- Network connectivity to Kiro instance
